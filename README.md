gateblu-ios
===========

Mobile gateway to manage BLE enabled devices and plugins in IOS

# Installation:

1. Install Xcode
2. Install Cocoapods
  * `sudo gem install cocoapods`
  * `pods setup`
3. Clone this repo
  * `git clone git@github.com:octoblu/gateblu-ios`
  * `cd gateblu-ios/gateblu`
  * `pod install`
4. Open `gateblu.xcworkspace`
